## 题目描述
不允许使用任何数学库，求给定任意正double浮点数的正开方根。精确到小数点后六位,四舍五入。输入有多组案例。  
**提示**：double类型的乘除和比较会有精度问题，比较大小和相等的时候请注意设置精度阀值。可尝试搜索 C++ double 精度 进行了解。

## 输入
```
2  
4.5  
3.14  
2.73  
```

## 输出
```
1.414214  
2.121320
1.772005
1.652271
```
